| **[Beyond the Prompt: Jailbreaking Function-Calling LLMs via Simulated Moderation Traces](https://arxiv.org/abs/2607.00481v1)** | 2026-07-01 | <details><summary>Show</summary><p>Jailbreak attacks remain a critical threat to the safe deployment of large language models (LLMs).
