| **[SelfGrader: LLM Jailbreak Detection via Anchored Token-Level Logits](https://arxiv.org/abs/2604.01473v3)** | 2026-05-28 | <details><summary>Show</summary><p>Large Language Models (LLMs) are powerful tools for answering user queries, yet they remain highly vulnerable to jailbreak attacks. To overcome these limitations, we propose SelfGrader, a lightweight guardrail method that formulates jailbreak detection as a numerical grading problem using anchored token-level logits. Specifically, SelfGrader evaluates the safety of a user query within a compact set of numerical tokens (NTs) (e.g., 0-9) and interprets their logit distribution as an internal safety signal.
