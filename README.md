| **[Good-Enough LLM Obfuscation (GELO)](https://arxiv.org/abs/2603.05035v1)** | 2026-03-05 | <details><summary>Show</summary><p>Large Language Models (LLMs) are increasingly served on shared accelerators where an adversary with read access to device memory can observe KV caches and hidden states, threatening prompt privacy for open-source models.
